## Getting Started <a name = "getting_started"></a>

### Prerequisites

You need to hace `GO` installed in your computer.

### Installing

```bash
$ git clone https://github.com/librity/nc_nomadcoin
$ cd nc_nomadcoin
$ go run main.go
```

Automagically analyze race conditions during compilation:

```bash
$ go build -race
```
